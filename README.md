# Placement Interaction System

A full-stack web application designed to manage and track student placements efficiently.  
This system provides separate dashboards and role-based features for **Students, Employers, Placement Officers, and Admins**.

---

## Features

### Student
- Browse and apply for job opportunities  
- View company details  
- Track application status  
- Manage profile & resume  

### Employer
- Post job openings  
- Manage job listings  
- Review student applications  
- Shortlist candidates  
- Schedule interviews  

### Placement Officer
- Manage placement records  
- Monitor student applications  
- Generate year/department-wise reports  
- Coordinate employerâ€“student interactions  

### Admin
- Manage users & roles  
- Approve companies  
- Maintain system settings  
- Oversee complete placement data  

---

## Tech Stack
- **Frontend:** HTML, CSS, JavaScript, React 
 

---

## Objectives
- Digitize the campus placement workflow  
- Reduce manual effort for placement officers  
- Provide a smooth job application experience for students  
- Allow companies to directly interact with student applicants
